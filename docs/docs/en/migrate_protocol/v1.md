# Migration Protocol v1

The migration record filename is `migration_{date}_{time}.json`, for example `migration_251201_0820.json`. The file content is roughly as follows:

```json
{
  "version": "1.0",
  "update": {
    "filename": { "hash": "hashstr" },
    "dirname": {
      "hash": "hashstr",
      "filename": { "hash": "hashstr" }
      //   ...
    }
  },
  "deleted": [
    "dirname",
    "filename"
    // ...
  ]
}
```
